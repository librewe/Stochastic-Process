## 问题1（正常返是类性质）
我们在课上证明了，如果一个马尔可夫链是不可约的，那么要么每个状态都是常返的，要么每个都不是。请证明对于正常返有相同的结论，即对不可约马尔可夫链，如果一个状态是正常返的，则每个状态都是正常返的。
**证明：** 约定用$T_{ij}^{k}$表示从状态 $i$ 开始第 $k$ 次转移到状态 $j$ 的步数，并简记 $T_{ij}^1 = T_{ij}$。下证：当 $\E{T_{ii}}<\infty$ 且存在一条从 $i$ 到 $j$ 的有限路径时，有$\E{T_{jj}}<\infty$。不妨这条路径是简单的，设这条路径上所有概率的乘积为 $q$。<!--路径长度为 $L$。 因 $i$ 常返，所以 $\Pr{T_{ji}<\infty}=1$ ，否则 $i$ 将有概率 $q(1-\Pr{T_{ji}<\infty})$ 先到达 $j$ 后不能返回。 --> 首先，<!--由强马尔可夫性，接下来出现的条件随机变量均等于随机变量。-->用 $T_{ij}^*$ 表示从状态 $i$ 转移到状态 $j$ 而不经过 $i$ 的步数，事件 $\mathscr{A}$表示 $i$ 回到自身前先经过$j$，$T_{ii}|\mathscr{A}=T_{ij}^*+T_{ji}$，$\E{T_{ii}} = \E{T_{ij}^*+T_{ji}}\Pr{\mathscr{A}}+\text{OtherCases}$，$\E{T_{ji}}\le \E{T_{ii}}/\Pr{\mathscr{A}}<\infty$，可得 $\E{T_{ji}}<\infty$。同理，$T_{ij}^*$ 的期望也是有限的。然后，设 $r=\Pr{\mathscr{A}}$ （$r$ 至少是 $q$），于是 $\E{T_{ij}}=\sum_{k=0}^{\infty} (1-r)^kr \E{(kT_{ii}+T_{ij}^*)}\le \frac{\E{T_{ii}}}{r}+\E{T_{ij}^*}<\infty$。最后，由于$T_{jj} \le T_{jj}|\{经过i\} = T_{ji} + T_{ij}$，于是 $\E{T_{jj}} \le \E{T_{ji}} + \E{T_{ij}} < \infty$。

## 问题2（破镜难重圆）
在宏观世界中，热力学第二定律告诉我们，孤立系统会自发地朝着熵增加的方向演化。例如，将所有的气体分子用隔板隔在一个盒子的左半边，撤去隔板后，分子会均匀地散布到整个盒子中。然而，如果在微观层面用马尔可夫链来描述这个过程，如果这个马尔可夫链是正常返的，经过足够长的时间，系统会以概率 $1$ 回到初始状态，这难道和热力学第二定律矛盾吗？

为了解答这个问题，我们考虑一个包含 $N$ 个粒子的简化模型，*埃伦费斯特瓮模型（Ehrenfest model）*。 假设有两个盒子 A 和 B，里面共有 $N$ 个粒子（$N\geq 2$）。在每一个离散的时间步，我们随机均匀地从这 $N$ 个粒子中挑出一个，把它移到另一个盒子里。令 $X_n$ 表示第 $n$ 步时盒子 A 中的粒子数，显然 $\{X_n\}$ 是一个状态空间为 $\{0, 1, \dots, N\}$ 的马尔可夫链。
1. 证明该马尔可夫链不可约并且是正常返的，并求出它的唯一平稳分布 $\pi$。
    **证明：** 该马尔可夫链显然不可约，因为任意状态可以到达相邻状态。注意到 $N$ 步内任意状态两两可达，设从状态 $j$ 出发走 $N$ 步经过 $i$ 的概率为 $q^{N}_{ji}$，显然$0<\min_{j} q^N_{ji}\triangleq q$，则 $\E{T_{ji}}=\sum_{n=0}^{\infty} \Pr{T_{ji}>n}\le\sum_{n=0}^{\infty}N\Pr{T_{ji}>nN}\le\sum_{n=0}^{\infty} N(1-q)^n \lt \infty$，于是 $\E{T_{00}}=1+\E{T_{10}}<\infty$，（而正常返是类性质）则该链是正常返的。平稳分布 $\pi$ 满足 $\pi P = \pi$，$P$ 是转移概率矩阵。由齐次性可先设 $\pi_0=1$ 求解后归一化得 $\pi(i) = \binom{N}{i} 2^{-N}$。
2. 假设初始状态 $X_0=N$，即所有粒子都在盒子 A 中。证明系统以概率 $1$ 会在有限步内回到状态 $N$，并计算首次返回状态 $N$ 的期望时间。
    **证明：** 以概率 $1$ 在有限步内回到状态 $N$ 即常返的定义，而1.已证正常返。首次返回时间期望 $\E{T^{(1)}_{NN}}=\cfrac{1}{\pi_N}=\frac{2^N}{\binom{N}{N}}=2^N$，因 $m_{ji}=1+\sum_{k\neq i}p_{jk}m_{ki}$，$\sum_{j}\pi_j m_{ji} = \sum_{j}\pi_j + \sum_{k\neq i}\sum_{j} \pi_j p_{jk} m_{ki}=1+\sum_{k\neq i} \pi_k m_{ki}\Rightarrow \pi_im_{ii}=1$，其中 $m_{ji}=\E{T^{(1)}_{ji}}$。
3. 假设每秒钟可以发生 100 亿次转移，计算当 $N=100$ 时，系统从 $X_0=N$ 出发，首次返回状态 $N$ 的期望时间。把这个期望时间与宇宙的年龄（约 $1.38 \times 10^{10}$ 年）进行对比，并解释为什么我们在现实中永远观察不到“破镜重圆”或“气体倒流”的宏观现象。
    **解：** $2^{100}/(100 \times 10^8) = 2^{100}/(10^{10}) \gt 2^{60}$ 秒，约为 $365.6$ 亿年，远超宇宙的年龄。虽然从理论上讲系统会回到初始状态，但实际几乎不可能观察到这种宏观现象。


## 问题3（林德利过程和系统稳定性）
在概率论中，*林德利过程（Lindley process）* 是一类定义在非负整数上的随机过程，在排队论、风险理论等领域都有着广泛的应用，其第 $n$ 步的状态 $X_n$ 可以由前一步状态 $X_{n-1}$ 和一个随机增量 $\xi_n$ 刻画：
    $$
        X_n = \max\{X_{n-1} + \xi_n, 0\}.
    $$
现在我们来考虑一个具体的林德利过程，假设初始状态 $X_0 = 0$，$\set{\xi_n}_{n>0}$ 是一列独立同分布的整数取值随机变量，代表系统在每一步不考虑边界 $0$ 的情况下试图移动的距离，并且假设 $\abs{\xi_n}<=100$，且 $\xi_n$ 的分布保证了该马尔可夫链是不可约且无周期的。假设 $\Var{\xi_n}>0$，并记其期望为 $\E{\xi_n} = \lambda$。

1. 我们先来分析上述条件下林德利过程的常返性，请证明：
     1. 考虑过程 $Y_n = \sum_{i=1}^n \xi_i$，在 $n\to \infty$ 时，$\frac{Y_n}{n}$ 以概率 $1$ 收敛到 $\lambda$。
        **证明：** 由独立同分布，且$\E{|\xi_1|}<\infty$，则可应用强大数定律得 $\frac{Y_n}{n} = \frac{\sum_{i=1}^n \xi_i}{n} \xrightarrow{a.s.} \E{\xi_1} = \lambda$。
     2. 当 $\lambda < 0$ 时，该马尔可夫链是正常返的。
        **证明：** $\Pr{Y_N=\sum_{n=1}^{N} \xi_n>0}=\Pr{Y_N-\E{Y_N}>-N\lambda}<\exp{(\frac{-2(N\lambda)^2}{N\cdot 200^2})}$
        $\E{T_{00}}=\sum_{N=0}^{\infty} \Pr{T_{00}>N}=\sum_{N=1}^{\infty} \Pr[0]{X_n > 0, \forall n \le N}$
        $=\sum_{N=1}^{\infty} \Pr{Y_n > 0, \forall n \le N}<\sum_{N=1}^{\infty} \Pr{Y_N > 0}$
        $\overset{\text{Hoeffding}}<\sum_{N=1}^{\infty} \exp{(\frac{-N\lambda^2}{20000})}<\infty$
        而正常返是类性质，则该链是正常返的。
        > Hoeffding好像需要指数矩存在，由有界保证。
     3. 当 $\lambda > 0$ 时，该马尔可夫链是瞬时的。
        **证明：** $\E{N_{00}}=\sum_{n=1}^{\infty} \Pr[0]{X_n = 0}\le\sum_{n=1}^{\infty} \Pr{Y_n \le 0}\overset{\text{与ii.对称地}}<\infty$
     4. **（附加题）** 当 $\lambda = 0$ 时，该马尔可夫链是零常返的。

2. 我们来学习一个林德利过程在调度算法中的具体应用。在无线通信网络中，基站通常需要使用有限的无线电频谱资源来并发服务多个用户的下行数据流。如何根据实时的队列拥塞情况以及不同用户的信道质量，动态调度传输资源以防止网络崩溃，是网络控制领域的核心问题之一。

	考虑一个包含两个队列 $Q1$ 和 $Q2$ 的基站下行链路。在每个时刻，队列 $Q1$ 和队列 $Q2$ 分别有 $A_1(t)$和 $A_2(t)$ 个新数据包到达，其到达包数的期望分别为 $\E{A_1(t)} = \lambda_1<\infty$ 和 $\E{A_2(t)} = \lambda_2<\infty$，且有 $0<\Var{A_1(t)}, \Var{A_2(t)}<\infty$。假设 $\{A_i(t)\}_{t\ge 0}$ 是独立同分布的随机变量序列，并且$A_i(t)$ 之间相互独立。

	由于队列 $Q1$ 距离基站较近，队列 $Q2$ 距离较远，基站在每个时间步只能在以下两种服务模式中选择其一：
	* 模式 1：只服务队列 $Q1$。可从 $Q1$ 处理至多 3 个数据包。
	* 模式 2：只服务队列 $Q2$。可从 $Q2$ 处理至多 2 个数据包.

	令 $S_1(t), S_2(t)$ 为时刻 $t$ 基站试图处理的两条队列的包数，$S_1(t)\in \set{0,3}$ 且 $S_2(t)\in \set{0,2}$，记两条队列 $t$ 时刻的长度分别为 $Q_1(t)$ 和 $Q_2(t)$，队列演化遵循 $Q_i(t+1) = \max(0, Q_i(t) - S_i(t)) + A_i(t)$。
	1. 请证明，无论基站采用何种调度算法，当 $\lambda_1/3 + \lambda_2/2 \ge 1$ 时，该系统的状态序列 $\{(Q_1(t), Q_2(t))\}_{t \ge 0}$ 构成的马尔可夫链不可能是正常返的，特别地，当 $\lambda_1/3 + \lambda_2/2 >1$ 时，请证明该马尔可夫链一定是瞬时的。
        **证明：** 构造 $M(t)=\max\{Q_1(t)/3+Q_2(t)/2,0\}$ 每步变化量 $\Delta(t)=A_1(t)/3+A_2(t)/2-1$，$\E{\Delta(t)}=\lambda_1/3+\lambda_2/2 - 1\ge0$，
        $M(0)=0, M(t)\ge\sum_{i=1}^t \Delta(i), \Var{\Delta(i)}\in(0,\infty)$
        假设是正常返的，则存在 $(Q_1,Q_2)$ 的平稳分布，$M$ 也应平稳，用 $\mu_i$ 表示状态 $i$ 处的期望转移 $\E{6M(t+1)-6M(t)|6M(t)=i}$，则由$\E{M(t+1)}=\E{M(t)}$ 有 $\sum_i \pi_i \mu_i = 0$，对任意 $i$，$\mu_i \ge 6\E{\Delta(t)}\ge 0$，考虑 $i=0$ 处，由于方差大于 $0$，必有概率发生正的转移，于是 $\mu_0 > 0$ 导致矛盾。
        关于瞬时性，对构造的 $M(t)$ 考虑与1.的ii.类似的放缩，问题来到怎么控制 $\sum_{n=1}^{\infty}\Pr{\sum_{i=1}^n \Delta(i)\le 0}$ 收敛。要是 $A_1(t),A_2(t)$ 也是有界的就好了，没招了。
        经过一番无意义的探索，我决定放弃这个思路。
        我最终与*王浩东* 同学探讨后取得了如下思路：由强大数定律，
        $\frac1n\sum_{i=1}^{n}\Delta(i)\xrightarrow{a.s.}\E{\Delta(t)}=\lambda_1/3+\lambda_2/2 - 1>0$
        $\sum_{i=1}^{n}\Delta(i)\xrightarrow{a.s.}\infty$ 而 $M(t)\ge \sum_{i=1}^t \Delta(i)$，则
        $M(t)\xrightarrow{a.s.}\infty$，而状态 $0$ 当仅当 $M(t)=0$，于是它是几乎必然不返回的，$\Pr[0]{T_0<\infty}=1-\Pr[0]{T_0=\infty}=0<1$，从而是瞬时的。
